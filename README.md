# IDF Patches Project for Tactical Assault VR
Important Instructions:
​This room is designed to add patches and suits of IDF special units and infantry to the game. Please note the differences between the platforms:
​For PC users: The mods are designed for you. You can replace files and enjoy the new content.
​For VR headset users: Unfortunately, in the headset version, it is not possible to change or replace system files, so the mods will not work.
